## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

This project contains:
* TypeOrm implementation
* Authentication flow with JWT Strategy
* User domain
* Task domain
* CRUD of Task entity

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
